**Apex Speech To Text**

Enable voice-based input in Oracle APEX using browser capabilities to convert speech into text for faster and more accessible data entry.

**Overview**

This project demonstrates how to integrate Speech-to-Text functionality into an Oracle APEX application.

Users can speak through a microphone, and the application automatically converts the speech into text and fills APEX page items in real time.

**Features**
  -Voice input using microphone
  -Real-time speech-to-text conversion
  -Multi-language support
  -Easy integration with APEX page items
  -Works on modern browsers
  
**Technologies Used**
  -Oracle APEX
  -JavaScript (Web Speech API)
  -HTML / CSS
  
**Use Case**
  -Faster data entry
  -Hands-free form filling
  -Improved accessibility
  -Ideal for notes, comments, and descriptions
  
**How It Works**
  1.User clicks the microphone button
  
  2.Browser captures the voice input
  
  3.Speech is processed and converted into text
  
  4.The converted text is populated into an APEX page item
  
**Limitations**
  -Depends on browser support (best in Chrome)
  -Requires microphone permission
  -Accuracy may vary based on accent and background noise
  
**Future Enhancements**
  -Voice-based commands for navigation
  -Automatic language detection
  -Integration with AI services
  -Voice history tracking
